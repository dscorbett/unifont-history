\doxysubsection{Detailed Description}
Create the Unicode Hangul Syllables block from component letters. 

This program reads in a \char`\"{}hangul-\/base.\+hex\char`\"{} file containing Hangul letters in Johab 6/3/1 format and outputs a Unifont .hex format file covering the Unicode Hangul Syllables range of U+\+AC00..U+\+D7\+A3.
